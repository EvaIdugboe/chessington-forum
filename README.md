# Chessington Forum

## Run locally

```bash
npm install
npm run dev
```

## Deploy on Vercel

1. Push this folder to a GitHub repository.
2. Sign in to Vercel with GitHub.
3. Import the repository.
4. Click **Deploy**.

## Notes

The registration form is currently a front-end form only.  
To collect real submissions, connect it to Google Forms, Tally, Formspree, Airtable, or another form backend.
